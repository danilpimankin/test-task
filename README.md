## Installation
Clone the repository and install the dependencies using the following command:
```
npm i
```

## Deployment
Fill in the .env file and use the command:
```
npx hardhat run scripts/deploy.ts --network polygon-mumbai
```
## Test Running
Running contract tests: 
```
npx hardhat test
```
